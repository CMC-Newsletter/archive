Our Public API has finally been merged into the Professional API, and they're now collectively called... CoinMarketCap API. Yes, we're creative!

• If you're still using the Public API for some of your apps, please be sure to update with new keys today. From free to enterprise, we have the right set of data for you, so be sure to sign up now!

• You can now find parent blockchain metadata for cryptocurrencies built on top of other platforms, such as tokens on Ethereum! They are now available on endpoints such as lists.

• We've made significant improvements to the price conversion tool: You can now make historical conversions for both crypto-to-crypto and crypto-to-USD down to the minute! All other fiat conversion pairs have 5-minute specificity.

• You can now access hourly OHLCV! Just use the time_period="hourly" and set the "interval" parameter to "hourly" or one of the new hourly interval options.

• Make use of the increased minutely rate limits on plans! Standard plan has been upgraded from 30 to 60 calls per minute, Professional plan has been upgraded from 60 to 90 calls per minute, and Enterprise plan has been upgraded from 90 to 120 calls per minute. More for everyone!

• Find more updates and fixes that we've been making over at Version History on the documentation.

We're happy to say that we've been delivering on many of the requests that you've asked for, and we remain dedicated to making the best-ever cryptocurrency API for you!

If you're already subscribed to our API, you'll automatically receive new features as they roll out. Again, if you're still on the Public API, be sure to update today to get more robust access to our trusted data going forward.
