It's our 6th birthday today! To celebrate, we are announcing some updates. It's our gift to you for your support over the years.
New plans and pricing
  1. If you're on the free Basic tier, you can now get live price conversions
  2. Pay 12% less for the Hobbyist plan, and get one month historical price conversions
  3. On the Startup plan, you can now access one month of crypto OHLCV data
  4. Standard plan users have been upgraded to 3 months of historical market data across all historical endpoints
Brand new endpoint #18
This new endpoint fetches daily historical crypto rankings from the past. We've made historical ranking snapshots available all the way back to 2013! 

Our Enterprise, Professional, and Standard tiers now get access to this brand new #18th endpoint.
Spotlight: Our new mobile apps built on the same API

Our recently-released mobile apps on iOS and Android are using the exact same APIs as you are! We released never-seen-before features using the same API data, such as candlestick charts, portfolio, side-by-side crypto comparison, beautifully-loaded sparklines, and more.

So far, it's already supporting hundreds of thousands of users a day. It's a great example of large-scale applications that we've been building our API to support. Check it out and let us know how you like them!
